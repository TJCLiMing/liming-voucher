# liming-voucher — 黎明教會付款申請單（代傳票）

真耶穌教會 黎明教會的付款申請單工具。前端填單 → 存進 Google Sheet → 列印 A4 代傳票 → 月底一鍵產生會計月報。

## 技術架構

| 層 | 技術 |
|---|---|
| 前端 | 單一 `index.html`（vanilla JS），部署到 GitHub Pages |
| 後端 | Google Apps Script Web App（`Code.gs`），提供 JSON API + 月報產生 |
| 資料 | Google Sheet（科目對照 + 申請單流水帳 + 會計月報分頁） |
| PDF | 瀏覽器列印 → 另存 A4 PDF |

前端與後端用 `text/plain` POST 溝通以避開 CORS preflight。

## 整體流程

```
申請人填單（GitHub Pages）
   → 存檔（或按列印自動先存檔）直接寫入當月分頁「會計支出115MM」
     ＝會計 xlsx 格式，零轉換
   → 列印 A4 代傳票（拾萬～元 位數格；領款人/申請人留空給實體簽名）
   → 月底會計直接下載 Google Sheet 即為最終 Excel
```

## 設計定案（2026-07-05 討論，同日改版：單一資料分頁）

- **單一資料分頁**：存檔直接寫入當月分頁 `會計支出115MM`（自動建立），
  **B～J 欄與會計 xlsx 完全相同**，不再有「申請單」流水帳與「產月報」步驟。
- **一張單可多科目**（參照會計 xlsx 11506 的 NO16）：
  - 同科目多筆明細 → 併成一列：摘要「、」串接、合計寫 `=a+b` 公式
  - 不同科目 → 各一列，**共用同一個 NO**
- **NO**：月內流水整數（1、2、3…）；前端顯示/紙本印 `11507-16`（民國年月-NO）。
  前端建議號若被同時段別人用走，存檔時自動遞補下一號。
- **列印自動存檔**：按列印時若未存檔或內容有改動，先存檔取得 NO 再印；
  空白單直接印（方便印手寫用空白單）。
- **簽名**：申請人（必填）輸入後只存紀錄，A4 列印的領款人簽名／申請人一律留空給實體簽名。
  前端不再有領款人欄位，Sheet 的領款人欄記申請人姓名。
- **已存單據清單**：前端「已存單據（查詢 / 重印）」按鈕列出本月存檔（依 NO 分組），
  點選載入表單即可重印；載入後直接列印不重複存檔，修改後列印則以新 NO 另存。
- **科目**：前端下拉用自訂科目名稱，「科目」分頁 B 欄放會計科目對照
  （如 `清掃薪資 → 6211薪資費用`）。存檔時映射成 `6331伙食費` 這種代號+名稱；
  未對照的列整列標黃提醒。
- **金額位數格**：最高「拾萬」（上限 999,999），超過前端跳紅字。不加「百萬」欄。
- 會計原始 xlsx（`會計格式現支115年度*.xlsx`）**不進 git**（.gitignore 已排除）。

## Google Sheet 結構

- 分頁 **「科目」**（設定）：A 欄科目名稱、B 欄會計科目（代號+名稱）。
  可用試算表選單「黎明教會 → 初始化科目分頁」建立預設 11 個科目與對照。
- 分頁 **「會計支出115MM」**（每月資料，自動建立）：
  - 標題列在第 2 列、B 欄起（比照會計 xlsx 版面）：
    `NO | 月 | 日期 | 科目代號 | 會計名稱 | 摘要 | 合計 | 領款人 | 備註 | 建立時間`
  - 領款人欄記「申請人」姓名（前端只填申請人，必填）；K（建立時間）是系統額外欄
  - 日期為民國文字（`115/07/05`）；會計名稱為 `=MID(E?,5,LEN(E?))` 公式
  - 最下方合計列（`=SUM(H3:H?)` + I 欄「合計」）由程式在每次存檔後維護
  - 備註欄（J）由出納/會計直接在 Sheet 上註記（如「06/20已給會計」）

## GAS 部署

1. Google Sheet → 擴充功能 → Apps Script，貼上 `Code.gs`
2. 部署 → 新增部署作業 → 網頁應用程式
   - 執行身分：**我**
   - 誰可以存取：**任何人**
3. 複製 `/exec` 網址 → 填進 `index.html` 的 `GAS_API_URL`
4. 重新整理試算表後會出現「黎明教會」選單（產月報、初始化科目）

## 給 Claude Code 接手的重點

- 前端邏輯都在 `index.html` 的 `<script>`：`state` 是資料模型（每列明細含科目），
  `renderSheet()` 畫 A4 預覽，`splitDigits()` 金額拆位數，`COLS` 定義位數欄，
  `dirty` 旗標控制「列印前自動存檔」（`printSheet()`）
- 後端 `Code.gs`：
  - `saveVoucher()`：依科目分組 → 寫入當月 `會計支出115MM` 分頁（會計格式）→
    補 MID 公式與合計列；NO 取月內最大 +1（`maxNo_`）
  - `getMonthSheet_()` 自動建月分頁（版面、欄格式）；`findSumRow_()` 找合計列
  - `initSubjectSheet()` 建預設科目對照，已有資料不覆寫
- 沒填 `GAS_API_URL` 走本機示範模式（預設科目、存檔印 Console），方便改 UI
- **改 Code.gs 後**：貼進 Apps Script → 部署 → 管理部署作業 → 新版本 → 部署
  （只有部署擁有者 lmf 帳號能按；網址不變）
